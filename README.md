# Net Time

This is an Ada 2012 library that implements the internet time protocol, [RTF3339](http://tools.ietf.org/html/rfc3339).

# Building

## GNAT

```bash
$ cd build/gnat
$ make
```

# Dependencies

Ada 2012 compiler.

## Tested with

FSF GNAT 7.3.0
